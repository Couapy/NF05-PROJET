Gestion des passagers d'un aeroport - NF05 projet
Structures de données
Liste des structures de données avec une brève description :
 CBagageComporte le numéro de ticket du bagage et si il a été embarqué
 CCommandePermet d'associer un numéro à une fonction pour la console
 CDateStructure de date, avec le jour, le mois et l'année
 CHoraireComporte un entier pour l'heure et pour les minutes
 CPassagerComporte le nom, prénom, la nationalité, le billet et des int bouléens pour le bon fonctionnement du programme
 CSiegeAssocie un numéro de rangée et de colonne pour une place dans l'avion
 CVolComporte le numéro de vol, la destination, le nombre de places libres, de places réservées, de colonnes et rangées (pour la taille de l'avion)
